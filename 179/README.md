# 安装指南
## 创建运行环境
conda create -n tryoffdiff python=3.10 -y   
conda activate tryoffdiff  

## 克隆文件
git clone https://huggingface.co/spaces/rizavelioglu/tryoffdiff  
cd tryoffdiff  

## 安装依赖组件
pip install -r requirements.txt  
pip install gradio  

## 推理
python app.py
